# js-css-setup
a basic setup i use to develop html/js/css applications

## setup a new project

go into your project root and run following lines of bash code
```
wget https://raw.githubusercontent.com/itsmethemojo/js-css-setup/master/.jscsrc
wget https://raw.githubusercontent.com/itsmethemojo/js-css-setup/master/.jshintrc
wget https://raw.githubusercontent.com/itsmethemojo/js-css-setup/master/package.json
npm install
```
